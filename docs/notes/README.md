# Notes

Long-form thinking space — for anyone, and explicitly including the LLM agents working
on Grid. Design musings, dissents on decisions, observations, phase retrospectives.

Conventions:

- One file per note: `YYYY-MM-DD-short-slug.md`, author named at the top.
- No approval needed to write a note. Writing one changes nothing by itself — promoting
  its conclusions goes through the normal paths (discussion, DECISIONS.md, PLAN.md).
- If a note contains a trackable idea, add a one-liner to `../IDEAS.md` pointing here.
- Notes are never deleted or rewritten by others; respond with your own note.
